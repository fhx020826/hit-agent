# 学生端模块说明

## 学生端首页原则
学生登录后自动进入学生学习台，仅展示学习支持相关模块。学生端不显示教师分析结果、作业统计总览或教师管理入口。

## 模块清单
### 1. 我的课程
- 页面：`/student/courses`
- 定位：持续展示后台模拟教务处已经为当前学生初始化好的选课结果
- 页面内容：
  - 已选课程列表
  - 任课教师
  - 学期
  - 教学班
  - 作业、AI 助教、讨论空间、资料、匿名反馈入口
- 重要规则：
  - 学生不是课程关系的创建者
  - 当前阶段不强调自行输入课程码或搜索加入课程
  - 如果当前账号没有选课关系，页面会明确提示联系管理员或教务处完成初始化

### 2. 课程专属 AI 助教
- 页面：`/student/qa`
- 能力：课程资料检索 + 大模型生成 + 多轮连续问答
- 提问模式：仅 AI、仅教师、AI + 教师
- 可选模型：由后端返回平台已配置的大模型列表
- 关系说明：AI 助教只在学生已选课程范围内加载知识上下文；若没有课程关系，则相关入口应显示清晰引导

### 3. 多模态提问
- 输入形式：文字、图片、文档、压缩包
- 使用建议：尽量同时补充文字说明，便于系统更准确理解问题
- 当前边界：部分格式可完整抽取文本，部分格式仅保存供教师查看

### 4. 学习问答记录
- 页面：`/student/questions`
- 能力：查看历史提问、AI 回答、教师回答、附件数量与回答状态
- 目录整理：支持根目录建文件夹、在文件夹内继续建子文件夹，形成多级目录
- 内容类型：同一目录下可同时存在子文件夹、学习问答记录、记事簿
- 记事簿能力：支持标题、正文、图片上传、最后更新时间展示
- 排序规则：默认按更新时间倒序显示，最近整理或修改的内容优先靠前
- 导航方式：支持面包屑路径、返回上一级，避免进入多级目录后迷路
- 作用：支持后续复习、资料沉淀和专题化整理

### 4.1 课堂共享资料
- 页面：`/student/materials`
- 能力：同步查看自己已选课程下教师当前共享的 PPT、讲义、图片、视频和补充材料
- 补充动作：可直接请求教师上传讲义资料，教师端会收到提醒
- 若教师正在发起课堂同步展示，可进入只读同步查看页，实时看到页码与批注

### 4.2 课程讨论空间
- 页面：`/student/discussions`
- 能力：进入自己已选课程对应的学习共同体空间，与同学、教师、AI 助教一起交流
- 支持：实名发言、匿名发言、文件消息、@AI 助教、关键词搜索、教师/AI 发言筛选
- 匿名边界：匿名消息只显示为“匿名同学”，不能通过检索反查真实身份

### 5. 薄弱点分析
- 页面：`/student/weakness`
- 能力：根据提问记录推断可能掌握较弱的知识点，并给出复习建议
- 说明：结果是温和的学习参考，不是能力定性结论；分析范围应限制在学生已选课程内

### 6. 作业任务中心
- 页面：`/student/assignments`
- 能力：查看作业、确认收到、上传提交、查看提交状态与 AI 初步反馈
- 限制：
  - 最终成绩和最终评语仍以教师判断为准
  - 学生只能看到和提交自己已选课程下的作业

### 7. 匿名课堂反馈
- 页面：`/student/feedback`
- 特点：自愿填写、可跳过、不显示给教师个人身份
- 区分：学生端做的是填写动作，教师端看的是匿名统计结果
- 关系说明：只能对自己已选课程提交匿名反馈

### 8. 个性化主题设置
- 页面：`/settings`
- 设置项：白天 / 夜间 / 护眼模式、主色、字体、皮肤、语言
- 入口：统一在右上角设置中心，不占首页卡片

### 9. 个人中心
- 页面：`/profile`
- 能力：查看和补充姓名、专业、班级、联系方式、简介、头像等信息
- 规则：所有作业、提问、反馈与本人账号绑定；匿名发言仅隐藏展示身份

## 生效边界总结
- 学生不能自己决定加入任意课程
- 学生不能查看未选课程的资料、作业、讨论空间和匿名反馈
- 学生不能向非选课教师发起课程提问
- AI 助手不是用来决定课程关系的，只在既有课程关系中辅助学习
