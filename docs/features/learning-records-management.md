# 学习问答记录管理

## 功能概览
学习问答记录模块已升级为学生个人学习资料库，支持：
- 多级文件夹
- 记事簿文本与图片
- 文件夹 / 记事簿 / 问答记录统一列表
- 按更新时间排序
- 面包屑导航与返回上一级

该模块与课程专属 AI 助教、提问历史、课程资料、个人中心兼容：
- `/student/qa` 继续负责即时提问
- `/student/questions` 负责归档整理与复盘
- 课程维度仍作为一级过滤条件
- 数据仍绑定学生本人账号，不跨用户共享

## 多级文件夹设计
- 根目录允许直接创建文件夹
- 进入任意文件夹后，可继续创建子文件夹
- 通过 `parent_folder_id` 形成层级关系
- 当前设置最大层级为 5 级，避免无限嵌套导致界面混乱
- 文件夹支持重命名
- 删除文件夹采用明确级联策略：
  - 默认不允许删除非空文件夹
  - 前端二次确认后以 `cascade=true` 方式执行级联删除
  - 级联删除会同时移除其下子文件夹、记事簿、记事簿图片和问答记录

## 记事簿能力
- 记事簿可创建在根目录，也可创建在任意文件夹内部
- 记事簿与问答记录是并列内容类型
- 当前支持：
  - 标题
  - 多段纯文本正文
  - 图片上传
  - 最后更新时间
- 后续扩展兼容：
  - 星标
  - 富文本
  - 问答记录转存为笔记

## 排序逻辑
- 当前目录列表默认按 `updated_at desc`
- 支持切换 `updated_at asc`
- 排序对象统一包括：
  - 子文件夹
  - 记事簿
  - 学习问答记录

## 更新时间规则
- 文件夹更新时间：
  - 重命名时更新
  - 新增 / 删除子文件夹时更新
  - 新增 / 编辑 / 删除记事簿时更新
  - 问答记录移动、收藏状态变更、教师回复变更、删除时更新
- 记事簿更新时间：
  - 标题修改
  - 正文修改
  - 图片上传 / 删除
- 学习问答记录更新时间：
  - 新建记录
  - 移动到文件夹
  - 收藏状态变化
  - 教师回复状态变化

## 权限规则
- 学习问答记录、文件夹、记事簿均属于学生个人空间
- 普通学生只能读取和操作自己的目录内容
- 教师端不展示学生个人目录
- 管理端若需运维扩展，应通过后台接口实现，不通过学生页面暴露
