# 个性化主题设置

## 当前能力
- 白天模式
- 夜间模式
- 护眼模式
- 主题主色：蓝、绿、紫、橙、灰
- 字体：标准、圆润、宋体文档风、等宽学习风
- 皮肤：简洁风、科技风、温和风

## 入口策略
- 不在首页放置独立“外观设置模块”卡片
- 统一从右上角“外观”快捷入口或“设置中心”进入
- 教师端与学生端使用同一套外观系统，数据按账号隔离

## 持久化方案
- 前端先写入 `localStorage`，保证切换立即生效
- 登录后通过 `/api/settings/me` 保存到后端
- 下次登录自动恢复上次选择

## 默认策略
- 默认模式：`day`
- 默认主色：`blue`
- 默认字体：`default`
- 默认皮肤：`clean`
