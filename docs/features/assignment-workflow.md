# 作业任务流程

## 当前闭环
1. 教师发布作业
2. 学生查看并确认收到
3. 学生上传一个或多个文件提交
4. 系统生成 AI 辅助反馈（如教师开启）
5. 教师查看已提交 / 未提交 / 已确认未提交 / 未确认收到名单

## 教师发布字段
- 作业标题
- 作业说明
- 所属课程
- 面向班级
- 截止时间
- 附件要求
- 提交格式要求
- 评分说明
- 是否允许补交
- 是否开启 AI 辅助反馈
- 提前提醒天数

## 当前限制
- 自动提醒天数已存储，但尚未挂接独立定时任务服务
- AI 反馈依赖文件解析质量，复杂版式文档可能仅给出保守建议
- AI 反馈不构成最终成绩
