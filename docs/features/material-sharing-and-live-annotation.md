# 教学资料共享与实时批注

## 1. 目标
形成“教师上传资料 -> 选择共享范围 -> 学生查看 / 请求 -> 教师发起课堂同步展示 -> 教师批注 -> 学生同步查看 -> 结束时选择保存或不保存”的完整链路。

## 2. 教师资料库
- 页面：`/teacher/materials`
- 支持上传 PPT / PDF / DOC / DOCX / 图片 / TXT / Markdown 等资料
- 支持查看资料列表、共享到学生端、发起课堂同步展示

## 3. 学生请求讲义
- 页面：`/student/materials`
- 学生可对当前课程发起“请求讲义 / 请求资料”
- 教师端收到站内提醒，并可将请求标记为 `approved / rejected / shared`

## 4. 课堂同步展示
- 教师可从资料库中选择资料并开始共享
- 当前共享状态会写入 `material_share_records`
- 学生端可通过课程页面进入只读同步查看页
- 同步内容包括：当前页码、批注轨迹、共享结束状态

## 5. 批注工具
当前版本支持以下批注工具语义：
- `pen`：钢笔
- `pencil`：铅笔
- `ballpen`：圆珠笔
- `highlighter`：荧光笔
- `flash`：速消笔

每条批注记录包含：
- 颜色
- 线宽
- 点位轨迹
- 是否临时笔迹
- 到期消失时间

## 6. 保存 / 不保存逻辑
- 结束共享并保存：只保存非临时笔迹，生成 `saved_annotation_versions`
- 结束共享不保存：删除本次共享下的批注记录
- 临时速消笔默认不进入永久保存版本

## 7. 实时同步
- 课堂展示页通过 WebSocket 连接 `/api/materials/live/{share_id}/ws`
- 服务端广播以下事件：
  - `share_started`
  - `page_changed`
  - `annotation_created`
  - `share_ended`

## 8. 权限边界
- 只有教师可发起共享和保存批注
- 学生只能查看共享内容，不能修改批注
- 未授权用户不能下载共享资料或查看实时共享状态
