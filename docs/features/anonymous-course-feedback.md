# 匿名课堂反馈

## 机制说明
- 教师端：触发问卷、查看汇总结果
- 学生端：自愿填写或跳过匿名反馈
- 教师不可见学生公开身份
- 教师端只显示统计结果与文本建议

## 当前实现
- 教师可手动创建问卷实例
- 学生通过 `/student/feedback` 查看待填写问卷
- 学生可提交或跳过
- 教师通过 `/teacher/feedback` 查看统计分析

## 统计字段
- 目标学生数
- 参与人数
- 参与率
- 评分题分布
- 选择题分布
- 开放文本建议

## 当前边界
- 自动按课程结束时间触发为预留能力
- 当前统计按整个平台学生数量做参与率计算，后续可进一步收缩到课程目标学生范围
