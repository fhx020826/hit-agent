# 产品总览

## 1. 产品定位
“面向前沿学科的智能教学平台”是一个围绕教师教学全过程打造的智能伙伴系统。平台不把 AI 定位为独立教学主体，而是定位为帮助教师完成课程设计、课堂实施支持、学生互动响应、作业反馈和教学优化复盘的提效工具。

## 2. 价值主张
### 2.1 教师侧
- 降低前沿课程备课门槛
- 提高旧 PPT、旧教案的更新效率
- 将学生提问、作业完成和匿名反馈收束到统一工作台
- 为形成性评价与课堂改进提供数据支撑

### 2.2 学生侧
- 获得课程专属 AI 助教支持
- 使用更自然的问答方式理解课程知识点
- 可选择 AI、教师或双通道提问
- 可在保护公开身份的前提下匿名发言或匿名反馈

## 3. 角色化界面策略
- 未登录：展示产品介绍与统一登录 / 注册入口
- 教师登录：进入教师工作台，仅展示教师功能模块
- 学生登录：进入学生学习台，仅展示学生功能模块
- 设置入口：统一放在右上角，不占首页教学主模块
- 管理员登录：进入教务模拟管理与用户管理视图，负责预置课程关系而非参与教学流程

## 3.1 教务模拟主线
- 平台当前采用后台模拟教务数据作为课程关系主线
- 管理员端统一初始化教师账号、学生账号、课程、授课关系与选课关系
- 教师登录后看到的是“我的授课课程”，学生登录后看到的是“我的课程”
- AI 助手只在既有课程关系中辅助教学与学习，不负责创建课程关系

## 4. 产品闭环
1. 管理员初始化演示教务数据，生成教师、学生、课程、授课与选课关系
2. 教师在“我的授课课程”中确认自己负责的课程
3. 教师进行课程设计、资料管理、作业发布和 AI 配置
4. 学生在“我的课程”中查看已选课程并进入问答、讨论、资料和作业流程
5. 系统在既有课程关系内生成 AI 辅助反馈与学习支持
6. 课程结束后教师触发匿名反馈，学生自愿填写
7. 教师查看反馈统计、问题回复情况和教学改进建议

## 5. 当前版本亮点
- 统一账号体系与角色化首页
- 后台模拟教务数据与稳定的课程关系初始化
- 教师端“我的授课课程”与学生端“我的课程”长期可见
- 右上角统一登录入口与设置中心
- 支持豆包、千问等 OpenAI 兼容大模型配置
- 多模态课程问答：文本 + 图片 + 文档 + 压缩包
- AI / 教师 / 双通道回答机制
- 课程讨论空间：学生、教师、AI 助教在同一课程班级空间内互动
- 讨论空间消息检索：支持关键词搜索、成员发言筛选与上下文定位
- 教学资料库 + 课堂同步展示 + 实时批注
- 管理员账号与全站用户管理
- 作业任务确认、提交与 AI 初评闭环
- 匿名课堂反馈与教师端统计分析

## 6. 比赛展示建议主线
- 教师智能课程设计与前沿内容融入
- 旧课件快速焕新
- 学生课后提问与教师闭环反馈
- 作业与匿名反馈驱动的教学优化

## 7. 当前真实边界
- 问卷自动定时触发尚未配置独立调度器
- 教师消息提醒当前为站内提醒，不含邮件 / 短信
- 部分文件格式仅支持上传保存和教师查看，AI 不保证深度解析
- AI 批改只提供辅助参考，不产生最终成绩
