# 作业辅助批改 Agent

## 模块名称
作业辅助批改 Agent

## 模块定位
在学生作业提交后生成结构、逻辑与规范性反馈参考，辅助教师快速判断作业完成质量。

## 核心功能
- 读取可解析提交文件
- 生成摘要
- 提供结构性反馈
- 提供逻辑性反馈
- 提供规范性建议

## 输入内容
- 作业标题与要求
- 学生提交的可解析文本

## 输出内容
- AI 初步反馈摘要
- 结构建议
- 逻辑建议
- 规范建议
- 评分参考提示

## 用户角色
教师主用，学生可查看部分开放结果。

## 权限范围
不能直接写入最终成绩。

## 典型场景
教师批阅前快速浏览学生作业质量。

## 数据来源
作业任务、学生提交文件、模型生成结果。

## 边界与限制
复杂版式文档解析有限；AI 反馈不是最终评分。

## 与其他模块关系
隶属于作业任务闭环，为教师工作台和学生作业中心提供参考结果。
