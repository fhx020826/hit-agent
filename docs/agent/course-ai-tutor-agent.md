# 课程专属 AI 助教 Agent

## 模块名称
课程专属 AI 助教 Agent

## 模块定位
服务学生课后学习支持，在教师设定边界内提供课程问答与复习引导。

## 核心功能
- 课时资料与课程包检索
- 大模型补充解释
- 多轮会话记忆
- 支持图片、文档、压缩包辅助提问
- 支持 AI / 教师 / 双通道回答模式

## 输入内容
- 学生问题文本
- 附件
- 课程 ID / 课程包 ID
- 目标回答模式
- 是否匿名发言
- 选定模型

## 输出内容
- AI 回答
- 回答依据来源
- 教师待回复状态
- 历史会话记录

## 用户角色
学生为主，教师可查看部分结果并补充回复。

## 权限范围
学生只能查看自己的问答记录；教师只能处理教师通道问题。

## 典型场景
学生围绕知识点理解、作业疑问、课件截图、资料文档发起学习提问。

## 数据来源
课程信息、课程包、上传附件、历史问答、模型服务。

## 边界与限制
部分附件类型当前仅支持留存和教师查看；解析深度依赖文件类型与模型能力。

## 与其他模块关系
与教师回复中心、薄弱点分析模块、课程资料模块紧密联动。
