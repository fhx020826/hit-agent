# 用户中心与个性化配置模块

## 模块名称
用户中心与个性化配置模块

## 模块定位
统一承载个人资料维护、外观设置与账号信息展示。

## 核心功能
- 维护教师 / 学生资料
- 管理主题模式、主色、字体和皮肤
- 展示角色信息与账号信息
- 提供退出登录入口

## 输入内容
- 用户资料编辑字段
- 外观配置字段

## 输出内容
- 最新资料信息
- 当前账号外观设置

## 用户角色
教师、学生

## 权限范围
每个账号只能修改自己的资料和设置。

## 典型场景
首次注册后完善资料、后续切换主题样式。

## 数据来源
`users`、`user_profiles`、`appearance_settings`

## 边界与限制
当前头像上传仍以路径字段预留为主。

## 与其他模块关系
为全站头部入口、角色识别、外观渲染提供基础数据。
