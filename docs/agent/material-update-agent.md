# 前沿内容融入与材料更新 Agent

## 模块名称
前沿内容融入与材料更新 Agent

## 模块定位
帮助教师在已有 PPT、教案、讲义基础上快速补充前沿内容、案例与拓展页结构。

## 核心功能
- 解析上传材料或正文
- 生成更新摘要
- 提供新增 PPT 页结构
- 生成配图建议

## 输入内容
- 课程 ID
- 更新任务标题
- 更新说明
- 材料文本或上传文件

## 输出内容
- 更新摘要
- 更新建议列表
- 草稿页列表
- 图片建议列表

## 用户角色
教师

## 权限范围
仅教师使用。

## 典型场景
教师希望把旧课件更新为更符合当前前沿学科发展趋势的展示版本。

## 数据来源
教师上传材料、课程上下文、模型生成结果。

## 边界与限制
当前不直接导出真实 PPT 文件，重点输出内容结构建议。

## 与其他模块关系
与课程设计模块共享课程上下文，可作为比赛展示中的“前沿融课”亮点。
