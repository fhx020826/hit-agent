# 教师智能课程设计 Agent

## 模块名称
教师智能课程设计 Agent

## 模块定位
服务教师课程准备阶段，为课程主题、授课对象、教学目标和前沿方向提供结构化设计建议。

## 核心功能
- 创建课程画像
- 生成课程包草稿
- 辅助确定教学目标与课堂活动方向

## 输入内容
- 课程名称
- 授课对象
- 学生层次
- 章节信息
- 教学目标
- 前沿方向

## 输出内容
- 课程记录
- 课程包记录
- 结构化教学建议

## 用户角色
教师

## 权限范围
仅教师可创建和维护课程数据。

## 典型场景
教师准备新课或重构现有课程章节内容时使用。

## 数据来源
课程基本信息、教师输入、课程包生成逻辑。

## 边界与限制
当前输出以结构建议和课程包内容为主，不直接替代教师完成最终教案。

## 与其他模块关系
为材料更新、AI 助教配置、学生问答等模块提供课程基础上下文。
