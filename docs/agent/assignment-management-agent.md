# 作业任务管理 Agent

## 模块名称
作业任务管理 Agent

## 模块定位
负责教师发布作业、学生确认收到与提交、教师查看未提交名单的流程组织。

## 核心功能
- 创建作业任务
- 记录学生确认状态
- 保存提交文件
- 汇总已提交与未提交名单

## 输入内容
- 教师发布的作业信息
- 学生确认与提交动作
- 班级信息

## 输出内容
- 作业记录
- 接收确认记录
- 提交记录
- 跟踪名单

## 用户角色
教师、学生

## 权限范围
教师管理作业；学生仅查看并提交自己的作业。

## 典型场景
课程布置课后任务、教师查看催交名单。

## 数据来源
作业表、确认表、提交表、学生资料表。

## 边界与限制
自动提醒调度尚未接入独立定时服务。

## 与其他模块关系
与作业辅助批改模块、个人中心、教学分析数据联动。
