# 匿名课堂反馈分析 Agent

## 模块名称
匿名课堂反馈分析 Agent

## 模块定位
汇总学生匿名问卷结果，为教师提供课程改进参考。

## 核心功能
- 创建问卷实例
- 汇总参与率
- 统计评分和选项分布
- 汇总匿名文本建议

## 输入内容
- 问卷模板
- 课程与课次信息
- 学生匿名反馈结果

## 输出内容
- 匿名统计分析
- 文本建议汇总

## 用户角色
教师查看分析，学生提交反馈。

## 权限范围
教师端不可查看学生公开身份。

## 典型场景
课程结束后进行匿名课堂反馈收集与复盘。

## 数据来源
问卷模板、问卷实例、问卷提交记录。

## 边界与限制
当前主要依赖教师手动触发问卷实例。

## 与其他模块关系
与教师工作台、教学分析仪表盘、学生反馈页面联动。
