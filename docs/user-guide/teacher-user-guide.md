# 教师用户手册

## 1. 登录与进入教师视图
- 点击右上角“登录”或“注册”
- 先选择“教师”角色
- 登录成功后自动进入教师工作台

## 2. 从右上角进入设置
- 点击右上角账号按钮
- 进入“个人中心”维护资料
- 进入“设置中心”切换主题与账号设置

## 3. 课程设计与材料更新
- 在“智能课程设计助手”中创建课程画像
- 生成课程包后，可继续到“PPT / 教案更新”上传旧材料
- 输入补充说明，生成更新建议与新增页草稿

## 4. 配置学生端 AI 助教
- 进入“课程专属 AI 助教配置”
- 设置范围规则、回答风格与开放能力

## 5. 管理作业
- 打开“作业任务管理”发布新作业
- 查看确认收到人数、未提交名单和已确认未提交名单
- 学生提交后查看 AI 初步反馈参考

## 6. 处理学生问题
- 打开“学生提问反馈中心”
- 查看站内提醒和问题详情
- 若学生上传附件，可直接下载查看
- 输入教师回复后提交，学生端会看到状态更新

## 7. 查看匿名课堂反馈
- 打开“匿名问卷分析”
- 手动触发问卷
- 读取参与率、评分分布与文本建议
