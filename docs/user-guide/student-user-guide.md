# 学生用户手册

## 1. 登录与进入学生视图
- 点击右上角“登录 / 注册”
- 选择“学生”角色
- 登录成功后自动进入学生学习台

## 2. 如何使用课程专属 AI 助教
- 打开“课程专属 AI 助教”
- 选择课程、课程包和模型
- 选择回答对象：AI、教师或 AI + 教师
- 输入文字问题，必要时上传图片、文档或压缩包

## 3. 如何匿名提问
- 提问前勾选“匿名发言”
- 匿名仅隐藏公开身份，不影响账号绑定和历史记录保存
- 教师端会显示“匿名学生”

## 4. 如何查看教师回复
- 在问答页或“学习问答记录”中查看
- 系统会标注 AI 回答、教师回复状态和时间戳

## 5. 如何上传作业
- 进入“作业任务中心”
- 先确认收到，再选择文件上传
- 若教师开放 AI 辅助反馈，可在提交后查看初步建议

## 6. 如何填写匿名反馈
- 进入“匿名课堂反馈”
- 可立即填写，也可选择稍后忽略
- 教师只能看到匿名统计结果

## 7. 如何修改外观
- 点击右上角“外观”或进入“设置中心”
- 调整主题模式、主色、字体和皮肤
